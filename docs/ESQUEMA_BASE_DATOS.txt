ESQUEMA DE BASE DE DATOS - TECHSTORE
====================================

BASE DE DATOS: TechStore.db
VERSIÓN: 1.0

TABLA: users
-----------
- id (INTEGER PRIMARY KEY AUTOINCREMENT)
- name (TEXT NOT NULL)
- email (TEXT UNIQUE NOT NULL)
- phone (TEXT NOT NULL)
- password (TEXT NOT NULL)

DESCRIPCIÓN: Almacena la información de los usuarios registrados en la aplicación.

TABLA: products
--------------
- id (INTEGER PRIMARY KEY AUTOINCREMENT)
- name (TEXT NOT NULL)
- description (TEXT)
- price (REAL NOT NULL)
- category (TEXT NOT NULL)
- stock (INTEGER NOT NULL)
- image (TEXT)

DESCRIPCIÓN: Almacena el catálogo de productos tecnológicos disponibles en la tienda.

CATEGORÍAS DE PRODUCTOS:
- Smartphones
- Laptops
- Tablets
- Audio
- Componentes
- Consolas

TABLA: cart
-----------
- id (INTEGER PRIMARY KEY AUTOINCREMENT)
- user_id (INTEGER NOT NULL) -> FOREIGN KEY (users.id)
- product_id (INTEGER NOT NULL) -> FOREIGN KEY (products.id)
- quantity (INTEGER NOT NULL)

DESCRIPCIÓN: Almacena los items del carrito de compras de cada usuario.

RELACIONES:
-----------
1. cart.user_id -> users.id (Muchos a Uno)
2. cart.product_id -> products.id (Muchos a Uno)

ÍNDICES:
--------
- users.email (UNIQUE) - Para búsqueda rápida de usuarios
- products.category - Para filtrado por categoría

OPERACIONES CRUD:
-----------------

USUARIOS:
- CREATE: insertUser(User user)
- READ: getUserByEmail(String email)
- UPDATE: (No implementado en esta versión)
- DELETE: (No implementado en esta versión)

PRODUCTOS:
- CREATE: insertProduct(Product product)
- READ: getAllProducts(), getProductById(int id)
- UPDATE: updateProduct(Product product)
- DELETE: deleteProduct(int id)

CARRITO:
- CREATE: addToCart(int userId, int productId, int quantity)
- READ: getCartItems(int userId), getCartTotal(int userId)
- UPDATE: updateCartQuantity(int cartId, int quantity)
- DELETE: removeFromCart(int cartId), clearCart(int userId)
